# Daily Brief Quality

Daily Brief v1 ranks local, user-owned daily-life data only. It does not invent calendar, email, or connector data.

Sections:
- Top 3 priorities
- Time-sensitive commitments
- Follow-ups due
- Meetings/situations to prepare for
- Waiting on others
- Risk items
- Low-effort admin tasks
- Deep-work tasks
- Stress/load note only when opted in
- Suggested next action
- Research/watch items
- Action proposals needing approval

Quality metrics:
- `briefRelevanceScore`
- `staleItemCount`
- `overdueItemCount`
- `acceptedSuggestionRate`
- `dismissedSuggestionRate`
- `confidence`

Feedback can be recorded per section. Feedback changes future scoring and review surfaces; it does not create sensitive profile facts automatically.
