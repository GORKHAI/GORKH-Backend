# Weekly Review

Weekly Review v1 summarizes confirmed local data and proposed daily-life items.

Sections:
- Completed tasks
- Missed commitments
- Open loops
- Follow-ups
- Stressful situations only when opted in
- Top wins
- Risks
- Next week preparation
- Suggested improvements

The weekly review does not diagnose stress, provide therapy, or make medical/legal/financial decisions. It is a planning and reflection surface, not an external action system.
