# Daily Brief Engine

Daily Brief generates a local planning summary from stored backend state.

Sections:

- Today's priorities
- Upcoming situations
- Open commitments
- Follow-ups
- Waiting on others
- Risk items
- Suggested next actions
- Stress/load note when opted in
- Research/watch items

The brief labels proposed items as reviewable planning candidates. It does not claim calendar/email access or fabricate external data.
